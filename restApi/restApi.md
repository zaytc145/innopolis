# Получение пользователей

```
GET - /users
```

| Код | Описание                    |
|-----|-----------------------------|
| 200 | Успешное выполнение запроса |
| 401 | Неверные учетные данные     |
| 403 | Отсутсвуют права доступа    |
| 500 | Ошибка сервера              |
